# Локальная библиотека контурных поз

`POSE_LINE_REFERENCE_LIBRARY` — необязательная локальная библиотека готовых
контурных рисунков взрослого тела. Она помогает выбирать и переносить позу, даже
когда пользователь не подключал `BODY_REFERENCE_LIBRARY`.

Менеджер также распознаёт уже собранную локальную коллекцию в
`BODY_REFERENCE_LIBRARY/02_LOCAL_ONLY/ANATOMY_CONTOUR_REFERENCE_LIBRARY/08_DOWNLOADED_LINE_ART`.
Она остаётся физически на прежнем месте, но получает ту же независимую мягкую роль
позы; повторно скачивать её не требуется.

## Граница влияния

Прошедший проверку лайн используется только как мягкий референс `POSE_SOFT`:

- положение суставов и линия действия;
- баланс и распределение веса;
- ракурс, укорочения и контакты;
- взаимное положение рук, ног и опоры.

Он не определяет телосложение, постоянные пропорции, лицо, стиль, одежду или
физиологию персонажа. В один вызов прикладывается не более одного такого лайна.
Если у персонажа есть каноническое тело, его пропорции остаются главными и
проверяются отдельно после генерации.

## Первичное развёртывание

Обычный запуск `scripts/bootstrap.ps1` ничего не скачивает автоматически. Если
библиотека отсутствует, он печатает `POSE_LINE_LIBRARY_OFFER` и команду, которую
ИИ должен предложить пользователю:

```powershell
# Пользователь явно разрешает сетевой сбор локальных кандидатов.
.\scripts\bootstrap.ps1 -CollectPoseLineLibrary
```

Для изменения глубины поиска можно указать `-PoseLinePages 1..10`.

## Полный цикл

```powershell
# 1. Собирает кандидатов из публичной поисковой выдачи в локальную игнорируемую папку.
.\.venv\Scripts\python.exe .\tools\pose_line_library_manager.py collect --pages 4

# 2. Строит контактные листы; их копии также архивируются в GENERATION_RESULTS.
.\.venv\Scripts\python.exe .\tools\pose_line_library_manager.py build-contact-sheets

# 3. После визуального просмотра заполните decision для каждой строки POSE_LINE_REVIEW.csv:
# PRIMARY_POSE_LINE | POSE_OR_DETAIL | STYLIZED_ONLY | REJECTED

# 4. Применяет только полный обзор и раскладывает неизменённые файлы по категориям.
.\.venv\Scripts\python.exe .\tools\pose_line_library_manager.py apply-review

# 5. Проверяет количество, роли, наличие файлов и SHA-256.
.\.venv\Scripts\python.exe .\tools\pose_line_library_manager.py validate
```

Автоматическая пиксельная фильтрация не является одобрением. Каждый кандидат
обязательно просматривается на контактном листе. Неполный `POSE_LINE_REVIEW.csv`
блокирует применение.

## Права и Git

Поисковая выдача может содержать Pinterest, фотостоки и авторские публикации.
Поэтому менеджер присваивает файлам статус
`UNVERIFIED_LOCAL_REFERENCE_ONLY` и сохраняет исходную страницу, прямую ссылку и
SHA-256. Изображения нельзя публиковать или добавлять в Git без отдельной проверки
лицензии. В репозитории хранятся только этот метод, инструкции и тесты.
